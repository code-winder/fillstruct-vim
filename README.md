# go-fillstruct-vim

forked from miya-masa/fillstruct-vim

## Requirement

* fillstruct

```
go get -u github.com/davidrjenni/reftools/cmd/fillstruct
go install github.com/davidrjenni/reftools/cmd/fillstruct
```

## install

* vim-plug
```
Plug 'code-winder/go-fillstruct-vim'
```

## usage

```
:GoFillStruct
```
